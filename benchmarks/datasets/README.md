# Benchmark Datasets

Downloaded 2026-04-08.

## SimpleQA

- **File**: `simple_qa_test_set.csv`
- **Source**: [openai/simple-evals](https://github.com/openai/simple-evals)
- **Download URL**: https://openaipublic.blob.core.windows.net/simple-evals/simple_qa_test_set.csv
- **Records**: 4326 factual questions with verified answers
- **Columns**: `metadata`, `problem`, `answer`

## TruthfulQA

- **File**: `TruthfulQA.csv`
- **Source**: [sylinrl/TruthfulQA](https://github.com/sylinrl/TruthfulQA)
- **Download URL**: https://raw.githubusercontent.com/sylinrl/TruthfulQA/main/TruthfulQA.csv
- **Records**: 790 questions testing common misconceptions (425 adversarial, 365 non-adversarial)
- **Columns**: `Type`, `Category`, `Question`, `Best Answer`, `Best Incorrect Answer`, `Correct Answers`, `Incorrect Answers`, `Source`
